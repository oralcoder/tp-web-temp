기공소 의뢰 승인시 알림 추가 안됨
배송등록 등 알림은 추가되지만 숫자는 업데이트 되지 않음

[기공소] 제작 완료되면, 납품 대기 상태
[기공소] 주문현황 관리 -> 납품대기 버튼 클릭 -> 배송 정보 입력 (배송업체 등 정보를 채우지 않으면 진행 안됨)
                    -> 상세 페이지 -> 납품대기 버튼 클릭 -> 배송 정보 입력 (배송업체 등 정보를 채우지 않아도 진행됨)
        - 입고확인 버튼을 클릭하면 배송정보가 출력됨(저걸 클릭하면 배송정보가 뜬다는걸 누가 알 수 있을까)

[치과] 기공소가 납품대기 버튼을 클릭하면 치과 화면에 입고확인 버튼이 생김. 
    - 기공소 납품하는 버튼은 납품대기가 아니라 납품 또는 배송이어야 할 것 같음
    - 치과 화면의 컬럼 이름은 입고확인이 아니라 진행상태가 되어야 함
    - 입고확인(입고대기가 되어야 함) 버튼 또는 상세페이지의 입고확인(상세페이지는 입고확인이 되어야 함) 버튼을 클릭 -> 정산대기
    - 정산대기 버튼 또는 상세페이지의 정산대기(정산완료 버튼이 되어야함) 버튼을 클릭 -> 세금계산서 발행 요청
    - 세금계산서 발행 요청(세금계산서 발행 대기가 되어야 함) 버튼 또는 상세페이지의 세금계산서 발행 요청 버튼을 클릭 -> 세금계산서 발행 요청 상태로 남아있음(세금계산서 발행이 되어야 함)
